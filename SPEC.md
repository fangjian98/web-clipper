# Web Clipper - 文章链接解析工具规范文档

## 1. 项目概述

- **项目名称**: Web Clipper
- **项目类型**: Web应用（单页应用）
- **核心功能**: 将任意网页链接解析为干净的Markdown格式，保留文章主体内容，去除无关元素
- **目标用户**: 需要保存网页文章为Markdown格式的用户（如笔记爱好者、研究人员、内容创作者）

## 2. 技术栈

- **前端框架**: React 18 + TypeScript
- **构建工具**: Vite
- **UI组件库**: TDesign（腾讯企业级设计系统）
- **样式方案**: CSS Modules + CSS变量
- **数据库**: Supabase（用于保存解析历史记录）
- **HTTP客户端**: Axios

## 4. 组件规范

### 4.1 Header组件

- 左侧：Logo图标 + 应用名称"Web Clipper"
- 右侧：主题切换按钮（可选）
- 高度：64px
- 背景：半透明毛玻璃效果

### 4.2 链接输入组件

- 输入框宽度：100%（最大800px）
- 占位符："请输入文章链接..."
- 右侧：解析按钮（主色调）
- 加载状态：按钮显示loading动画

### 4.3 解析结果区（双列）

- **左列 - 原文预览**
  - 显示提取后的文章内容（去除广告、导航等）
  - 纯展示，不可编辑
  - 滚动区域，最大高度500px

- **右列 - Markdown结果**
  - 可编辑的textarea
  - 支持语法高亮（可选）
  - 右上角：复制按钮、下载按钮

### 4.4 历史记录组件

- 卡片式列表
- 每条记录包含：标题、来源URL、解析时间、操作按钮
- 支持删除历史记录
- 分页或无限滚动

## 5. 功能规范

### 5.1 链接解析功能

1. 用户输入URL并点击解析
2. 后端服务提取网页内容（使用Jina AI Reader API或其他方案）
3. 提取正文，去除导航、页脚、广告、侧边栏
4. 转换为Markdown格式
5. 展示解析结果
6. 自动保存到历史记录

### 5.2 下载功能

- 下载为.md文件
- 文件名：文章标题或时间戳
- 编码：UTF-8

### 5.3 历史记录功能

- 保存URL、标题、Markdown内容、解析时间
- 支持查看历史记录详情
- 支持删除单条记录
- 支持复制历史记录内容

### 5.4 Supabase数据库表设计

```sql
-- 解析历史记录表
create table clip_history (
  id uuid default gen_random_uuid() primary key,
  url text not null,
  title text,
  content text,
  created_at timestamp with time zone default timezone('utc'::text, now())
);
```

## 6. 页面响应式

- **移动端** (< 768px): 单列布局，历史记录折叠展示
- **平板** (768px - 1024px): 双列紧凑布局
- **桌面** (> 1024px): 双列完整布局
